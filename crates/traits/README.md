# Traits for Roxy

[![Crates.io](https://img.shields.io/crates/v/roxy-traits.svg)](https://crates.io/crates/roxy-traits)
[![Docs.rs](https://docs.rs/roxy-traits/badge.svg)](https://docs.rs/roxy-traits)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Core trait definitions for Roxy components including Backend, Cache,
`RateLimiter`, `LoadBalancer`, and runtime abstractions for spawning,
clocks, and metrics.
